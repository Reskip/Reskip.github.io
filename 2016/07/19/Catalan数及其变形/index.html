Catalan数

Catalan number，是一个在组合数学技术问题中常出现的数列，特别在计算机科学方面有广泛应用。

前20项为（OEIS中的数列A000108）：1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862, 16796, 58786, 208012, 742900, 2674440, 9694845, 35357670, 129644790, 477638700, 1767263190

其中 数列的第n项记为C[n]

Catalan数可解问题

1.表示所有包含n组括号的合法运算式的个数

2.一个有n个X和n个Y组成的字串，且所有的部分字串皆满足X的个数大于等于Y的个数

3.有n+1个叶子的二叉树的个数 //由于有根节点

4.n个元素合法的入栈出栈次序

5.用n个长方形填充一个高度为n的阶梯状图形的方法个数

6.通过连结顶点而将n + 2边的凸多边形分成三角形的方法个数

公式推导及求解

以问题4为例 //所有问题可以一一对应互相转化
由于至少有一个时刻（除去开始），使得此时栈内为空。假设栈内为空时已入栈、出栈k次，则在此条件下所有的方法数位C[k]*C[n-k+1]，即将一个大问题分解成两个小问题去解决。

即得到一递归式：
C[n] = C[0] C[n-1] +C[1] C[n-2] + … + C[n-1] * C[0]

化简得 $C[n] = (2n)!/(n+1)!(n)!$
或递推式 $C[n+1]=C[n]*2(2n+1)/(n+2)$

经典题型

HDU 2067 小兔的棋盘

Problem Description
小兔的叔叔从外面旅游回来给她带来了一个礼物，小兔高兴地跑回自己的房间，拆开一看是一个棋盘，小兔有所失望。不过没过几天发现了棋盘的好玩之处。从起点(0，0)走到终点(n,n)的最短路径数是C(2n,n),现在小兔又想如果不穿越对角线(但可接触对角线上的格点)，这样的路径数有多少?小兔想了很长时间都没想出来，现在想请你帮助小兔解决这个问题，对于你来说应该不难吧!

Input
每次输入一个数n(1<=n<=35)，当n等于－1时结束输入。

Output
对于每个输入数据输出路径数，具体格式看Sample。

Sample Input
1
3
12
-1

Sample Output
1 1 2
2 3 10
3 12 416024

注意格式，求得 2 * C[n] 即可

Catalan数的变形

通常的Catalan组合数问题求解的是例如n个元素的入栈出栈问题，即n个元素全部入栈，全部出栈的方法数

变形问题例如：

1.进行N次入栈、M次出栈操作 (N>=M) ，合法的操作种数

2.N男M女 (N>=M) 站队，求在任一位置之前的女生不多于男生的排列种数

3.N*M的方格地图，求从(0,0)到达(n,n)不越过x=y这条线的走法

即存在两个不同参数N和M的Catalan问题 //原问题可看成此问题在N=M情况下的特解

思路类似于经典Catalan证明方式
N+M个位置 选择 N个位置放置出栈 即 C(m+n,m)
然后不满足条件的情况 任意不满足条件 即 存在一个位置此时有 K 个入栈和 K+1 个出栈

则，其后必然有 N - ( K + 1 ) 个出栈及 M - K 个入栈

化简既有：
$C[M,N] = C(m+n,m)-C(m+n,m+1)$

例题：
HDU 1133 Buy the Ticket

Problem Description
The “Harry Potter and the Goblet of Fire” will be on show in the next few days. As a crazy fan of Harry Potter, you will go to the cinema and have the first sight, won’t you?

Suppose the cinema only has one ticket-office and the price for per-ticket is 50 dollars. The queue for buying the tickets is consisted of m + n persons (m persons each only has the 50-dollar bill and n persons each only has the 100-dollar bill).

Now the problem for you is to calculate the number of different ways of the queue that the buying process won’t be stopped from the first person till the last person.
Note: initially the ticket-office has no money.

The buying process will be stopped on the occasion that the ticket-office has no 50-dollar bill but the first person of the queue only has the 100-dollar bill.

Input
The input file contains several test cases. Each test case is made up of two integer numbers: m and n. It is terminated by m = n = 0. Otherwise, m, n <=100.

Output
For each test case, first print the test number (counting from 1) in one line, then output the number of different ways in another line.

Sample Input
3 0
3 1
3 3
0 0

Sample Output
Test #1:
6
Test #2:
18
Test #3:
180

求 C[m,n] 注意，当m和n较大时需要学习大组合数取模的方法
